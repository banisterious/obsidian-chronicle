ScribeFlow Export Dropdown Mockup

Context: This mockup shows the export dropdown as it would appear in the dashboard controls section, positioned after the search functionality. The dropdown uses Lucide icons and provides clear visual hierarchy for the three export options.

Dashboard Controls with Export Button

Export Dropdown (Expanded State)

Copy as Markdown Table
Copy formatted table to clipboard
Save as CSV File
Export data for Excel or analysis
Save as JSON File
Export structured data with metadata
Design Notes:
  • Icons: Using Lucide icons for consistency with Obsidian's design language
  • Layout: Dropdown aligned to the right edge of the button for better screen edge handling
  • Descriptions: Each option includes a helpful subtitle explaining the use case
  • Visual Hierarchy: Clear distinction between title and description text
  • Hover States: Subtle background changes on hover for better UX
  • Button State: Chevron changes from down to up when dropdown is open

Responsive Behavior

Mobile/Narrow View: Controls stack vertically while maintaining usability and visual hierarchy.
